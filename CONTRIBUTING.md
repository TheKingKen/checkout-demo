# Contributing to checkout-demo

Thank you for your interest in this project!

## Project Scope

This repository is a **Node.js demonstration project** for Checkout.com's Hosted Payments Page (HPP) integration. It is designed to showcase payment integration patterns using JavaScript/Node.js.

## Questions and Support

### Questions About This Demo
If you have questions specifically about this Node.js/JavaScript checkout demo implementation, please:
- Open an issue in this repository
- Provide details about your environment and what you're trying to achieve
- Include relevant code snippets and error messages

### Questions About Checkout.com Java SDK
If you have questions about the **Java SDK** (checkout-sdk-java), please visit:
- Repository: [checkout/checkout-sdk-java](https://github.com/checkout/checkout-sdk-java)
- File issues there for Java-specific questions

### Questions About Other Checkout.com SDKs
Checkout.com provides SDKs for various languages. For questions about other implementations:
- **.NET SDK**: [checkout/checkout-sdk-net](https://github.com/checkout/checkout-sdk-net)
- **PHP SDK**: [checkout/checkout-sdk-php](https://github.com/checkout/checkout-sdk-php)
- **Python SDK**: [checkout/checkout-sdk-python](https://github.com/checkout/checkout-sdk-python)
- **Go SDK**: [checkout/checkout-sdk-go](https://github.com/checkout/checkout-sdk-go)

### General Checkout.com API Questions
For general API questions not specific to any SDK:
- Visit the [Checkout.com Developer Portal](https://www.checkout.com/docs)
- Check the [API Reference Documentation](https://api-reference.checkout.com/)
- Contact Checkout.com support directly

## Contributing Code

If you'd like to contribute improvements to this demo:
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request with a clear description of your changes

Thank you for helping to improve this project!
